## Test Practice

</br>

> ### 환경설정
>
> </br>

1. `npm init --yes` </br>
2. `npm install jest --global`</br>
3. `jest --init`</br>
4. `npm install--save-dev jest`

</br>

> ### Test 시작
>
> `npm run test`

</br>

> ### 파일이 변경 될 때 마다 테스트 코드 실행
>
>  </br>

`package.json 파일 내에`

 </br>
 
 ```
  "scripts": {
		"test": "jest --watch"
	},
 ```

 </br>
